# 🎭 Theater Frontend Design Project

A responsive frontend web application built using HTML, CSS, and JavaScript.  
This project focuses on UI design for a theater/ticket booking experience.

---

## 🚀 Features
- Fully responsive design (mobile + desktop)
- Multiple web pages layout
- UI components for tickets, shows, and gallery
- Clean and modern interface
- Interactive frontend elements

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla)  

---

## 📱 Project Type
Frontend UI Design Project (No backend)

---

## 🎯 Purpose
This project was built to improve frontend development skills, focusing on layout design, responsiveness, and user interface structuring.
